Problem Statement

This project focuses on developing a deep learning-based solution to classify aerial images into two categories: Bird and Drone, and optionally perform object detection to accurately locate and label these objects in real-world scenarios.

The solution is designed to support applications in security surveillance, wildlife protection, and airspace safety, where distinguishing between birds and drones is critical.

The project involves:

Building a custom CNN-based classification model
Applying transfer learning techniques
Optionally implementing YOLOv8 for real-time object detection
Deploying the final solution using Streamlit for interactive use
Real-World Applications
Wildlife Protection
Detect bird activity near wind farms or airport zones to prevent accidents
Security and Defense Surveillance
Identify unauthorized drones in restricted airspace and trigger alerts
Airport Bird-Strike Prevention
Monitor runways to detect bird presence and reduce collision risks
Environmental Research
Track bird populations using aerial imagery without misclassification
